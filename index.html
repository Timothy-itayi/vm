LC-3 Virtual Machine with JavaScript

This is a JavaScript implementation of the LC-3 virtual machine, an educational computer architecture.

Controls:

Any keys that are pressed while the terminal is focused will be forwarded to the VM.

On mobile (or devices with a touch screen), swiping on the terminal will simulate WASD key presses. To send keys other than this, you can tap anywhere in the terminal to manually focus the terminal and open a keyboard.

More Information

This project is inspired by the work of rpendleton and based on a tutorial written by Justin Meiners and Ryan Pendleton. If you're interested in writing your own VM (or just want to learn more about the project), you can find the tutorial here.
