About This Project

This project emerged from my journey learning about virtual machines and computer architecture through Justin Meiners and Ryan Pendleton's excellent tutorial. It demonstrates how a simple virtual machine can execute programs, showcasing fundamental concepts of computer architecture in an interactive way.

Interaction Guide

Desktop Usage

Use your keyboard to interact with the programs. All keystrokes are automatically forwarded to the virtual machine when the terminal is focused.

Mobile Usage

Swipe gestures simulate WASD movement. Tap the terminal area to bring up the keyboard for additional input options.
